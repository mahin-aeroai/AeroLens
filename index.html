Database Schema
Oracle DB — 13 Tables, 60+ Columns
Normalised schema with sequences, foreign keys, 15 indexes, and 2 materialised views for dashboard KPIs.
MST_COMPANY
  • COMPANY_ID (PK)
  • COMPANY_NAME, TYPE
  • HEADQUARTERS, WEBSITE
  • ANNUAL_REVENUE, EMPLOYEES
  • CERTIFICATIONS, GET_PROGRAM
  • CAREERS_URL, NOTES
CRM_CONTACT
  • CONTACT_ID (PK)
  • COMPANY_ID (FK)
  • CONTACT_NAME, DESIGNATION
  • EMAIL, MOBILE, LINKEDIN
  • CONTACT_STATUS
  • LAST_COMM_DATE
BIZ_OPPORTUNITY
  • OPP_ID (PK)
  • COMPANY_ID, CONTACT_ID (FK)
  • RFQ_NUMBER, TENDER_NUMBER
  • OPP_VALUE, PROBABILITY_PCT
  • STATUS, SUBMISSION_DATE
  • EXPECTED_AWARD_DATE
JOB_POSTING
  • JOB_ID (PK)
  • COMPANY_ID (FK)
  • POSITION_TITLE, DEPT
  • SALARY_MIN, SALARY_MAX
  • STATUS, JOB_TYPE
  • MY_APP_STATUS (personal)
TECH_PROFILE
  • TECH_ID (PK)
  • COMPANY_ID (FK, 1:1)
  • USES_AI, USES_ML
  • USES_DIGITAL_TWIN
  • CAD_TOOLS, SIM_TOOLS
  • CERTIFICATIONS_SW
SUP_SUPPLIER
  • SUPPLIER_ID (PK)
  • PRODUCTS_OFFERED
  • CERTIFICATIONS
  • APPROVAL_STATUS
  • AUDIT_STATUS
  • LAST_AUDIT_DATE
PRJ_PROJECT
  • PROJECT_ID (PK)
  • COMPANY_ID (FK)
  • PROJECT_NAME, SECTOR
  • START_DATE, END_DATE
  • PROJECT_VALUE
  • MILESTONE_1/2/3
DOC_DOCUMENT
  • DOC_ID (PK)
  • COMPANY_ID, OPP_ID (FK)
  • DOC_TYPE, DOC_TITLE
  • FILE_PATH, MIME_TYPE
  • VERSION, EXPIRY_DATE
  • IS_CONFIDENTIAL
CRM_ACTIVITY
  • ACTIVITY_ID (PK)
  • COMPANY_ID, CONTACT_ID (FK)
  • ACTIVITY_TYPE (CALL/MEET)
  • SUBJECT, OUTCOME
  • NEXT_ACTION_DATE
  • REMINDER_DATE
ERP_USERS / ROLES
  • USER_ID, ROLE_ID (PK)
  • USERNAME, EMAIL
  • PASSWORD_HASH (BCrypt)
  • LAST_LOGIN
  • ROLE_NAME, ROLE_DESC
  • IS_ACTIVE
ERP_MODULE_PERMS
  • ROLE_ID + MODULE_CODE
  • CAN_VIEW, CAN_CREATE
  • CAN_EDIT, CAN_DELETE
  • CAN_EXPORT
  • 13 modules × 5 roles
AUD_LOG
  • LOG_ID (PK)
  • TABLE_NAME, RECORD_ID
  • ACTION (INSERT/UPDATE)
  • OLD_VALUES (CLOB JSON)
  • NEW_VALUES (CLOB JSON)
  • CHANGED_BY, IP_ADDRESS
REF_LOOKUP
  • LOOKUP_TYPE, CODE
  • Company types, segments
  • Opportunity statuses
  • Job statuses
  • All dropdown values
  • Seeded on install

Development Roadmap
7-Phase Build Plan — 16 Weeks
From Oracle schema to full AI-powered intelligence platform.
Phase 1 · Weeks 1–3
Foundation
Oracle schema + .NET 8 setup + JWT auth
  • 13-table DDL + seed data
  • JWT login + refresh
  • Swagger UI
  • Role permissions seeded
Phase 2 · Weeks 4–5
Company Master
Core module + frontend wiring
  • CRUD + pagination + filters
  • Grid + list view
  • Tech profile upsert
  • CSV export
Phase 3 · Weeks 6–7
Contacts & CRM
CRM activity + reminder engine
  • Contact CRUD
  • Activity log (call/email/meeting)
  • 7-day reminder view
  • Follow-up timeline
Phase 4 · Weeks 8–9
BD Pipeline
Opportunities + Projects + Kanban
  • Opportunity Kanban board
  • Pipeline funnel chart
  • Project milestone tracker
  • Win rate analytics
Phase 5 · Weeks 10–11
Jobs & Suppliers
Career tracking + vendor management
  • Personal application tracking
  • Supplier audit workflow
  • Document upload + storage
Phase 6 · Weeks 12–13
Analytics & Reports
Live charts + Excel/PDF reports
  • Dashboard KPIs from Oracle views
  • 8 pre-built reports
  • NPOI Excel + iTextSharp PDF
Phase 7 · Weeks 14–16
AI Integration
GPT-4o + ML models
  • Company URL summarizer
  • NLP job-skill matcher
  • Opportunity scoring (XGBoost)
  • Contract risk analyser
